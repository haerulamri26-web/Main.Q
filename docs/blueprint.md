# **App Name**: HTML Reader

## Core Features:

- HTML Code Display: Display HTML code in a well-formatted and syntax-highlighted manner.
- URL Rendering: Render content available at an entered URL in a sandboxed iFrame.
- Live Editing: Allow users to edit the HTML code directly within the interface.
- AI Assistant Tool: Provide suggestions, automatic error checking, and code completion powered by generative AI.

## Style Guidelines:

- Primary color: Soft purple (#A78BFA) for a modern and calming feel.
- Background color: Light gray (#F5F5F5) to ensure readability.
- Accent color: Teal (#4FD1C5) for interactive elements.
- Body and headline font: 'Inter' sans-serif for a clean and modern look.
- Code font: 'Source Code Pro' monospaced font for clear code snippets.
- Use simple, consistent icons for file actions.
- Split-screen layout with code editor on one side and rendered output on the other.
- Subtle transitions for code updates and display changes.
